# Лабораторная работа №3. Очередь задач: итераторы и генераторы

## Обзор проекта

Данная работа реализует очередь задач `TaskQueue`, которая поддерживает итерацию, повторный обход, ленивую фильтрацию и потоковую обработку задач.

## Ключевые архитектурные решения

1. **Протокол итерации**: `TaskQueue` можно использовать в `for`, `list()` и других стандартных конструкциях Python.
2. **Повторный обход**: очередь не исчерпывается после первого прохода.
3. **Ленивая фильтрация**: фильтры по статусу и приоритету возвращают генераторы.
4. **Потоковая обработка**: источники задач отдают данные через генераторы без лишнего хранения в памяти.
